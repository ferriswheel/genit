Welcome to Genit !

Genit is a framework to build a static web site, that is a web site without server side programing language and database. A genit site consists only of xhtml code (+ css, medias and eventually javascript).

Genit is written in Ruby but there is no needs to know the Ruby language.

The Genit project is hosted on GitHub under the terms of the MIT license. Feel free to fork it. Each new release will be announced on the freshmeat site. The development progress will be announced on Twitter:
